A TerrainRegion is a snapshot of [Terrain](/docs/reference/engine/classes/Terrain) retrieved from the
[Terrain:CopyRegion()](/docs/reference/engine/classes/Terrain#CopyRegion) method. Can be later pasted into the Terrain
using [Terrain:PasteRegion()](/docs/reference/engine/classes/Terrain#PasteRegion).
