Engine Class

DoubleConstrainedValue

Deprecated

An instance which is used to create a number value which can never be less
than the MinValue or more than the MaxValue.

The DoubleConstrainedValue object has been deprecated as developers can now
use the [math.clamp()](/docs/reference/engine/libraries/math#clamp) function to constrain values.
