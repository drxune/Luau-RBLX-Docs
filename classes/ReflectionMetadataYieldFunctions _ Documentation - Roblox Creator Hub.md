Engine Class

ReflectionMetadataYieldFunctions

Acts as a [Folder](/docs/reference/engine/classes/Folder) containing information about yielding functions for a
[ReflectionMetadataClass](/docs/reference/engine/classes/ReflectionMetadataClass), and should be parented to it.
[ReflectionMetadataMembers](/docs/reference/engine/classes/ReflectionMetadataMember) that are parented
to this object will be perceived as yielding functions under the class this is
parented to.
