The SelectionLasso class is an abstract class of which the inheritors are able
to be attached to an object of the [Humanoid](/docs/reference/engine/classes/Humanoid) class or the
[BasePart](/docs/reference/engine/classes/BasePart) class. They can also be attached to a point in the
tridimensional space indicated by a Vector3 value.
