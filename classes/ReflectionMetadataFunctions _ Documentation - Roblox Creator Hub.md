ReflectionMetadataFunctions

Acts as a [Folder](/docs/reference/engine/classes/Folder) containing information about functions for a
[ReflectionMetadataClass](/docs/reference/engine/classes/ReflectionMetadataClass), and should be parented to it.

to this object will be perceived as functions under the class this is parented
to.
