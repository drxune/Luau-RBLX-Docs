A special type of [Pages](/docs/reference/engine/classes/Pages) object whose pages contain
[DataStoreObjectVersionInfo](/docs/reference/engine/classes/DataStoreObjectVersionInfo) instances from a [GlobalDataStore](/docs/reference/engine/classes/GlobalDataStore).
[Pages:GetCurrentPage()](/docs/reference/engine/classes/Pages#GetCurrentPage) can be used to retrieve an array of the
[DataStoreObjectVersionInfo](/docs/reference/engine/classes/DataStoreObjectVersionInfo) instances.
