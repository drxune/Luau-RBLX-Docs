Engine Class

SelectionPartLasso

Deprecated

An instance used to display a "lasso" between a [Humanoid](/docs/reference/engine/classes/Humanoid) Torso and a
[BasePart](/docs/reference/engine/classes/BasePart).

SelectionPartLasso has been deprecated. Developers are advised to use the
[Beam](/docs/reference/engine/classes/Beam) object instead.
