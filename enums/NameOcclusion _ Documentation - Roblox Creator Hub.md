Engine Enum

NameOcclusion

---

The NameOcclusion Enum is used to set the [Humanoid.NameOcclusion](/docs/reference/engine/classes/Humanoid#NameOcclusion)
property.

Items

| Name | Value | Summary |
| --- | --- | --- |
| NoOcclusion | 0 | Do not occlude (hide) any humanoid names. |
| EnemyOcclusion | 1 | Occlude (hide) all enemy humanoid names. |
| OccludeAll | 2 | Occlude (hide) all humanoid names. |
