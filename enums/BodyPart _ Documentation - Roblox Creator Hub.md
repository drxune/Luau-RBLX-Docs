Engine Enum

BodyPart

---

The BodyPart Enum determines which BodyPart a [CharacterMesh](/docs/reference/engine/classes/CharacterMesh) affects.
(the [CharacterMesh](/docs/reference/engine/classes/CharacterMesh) is used with R6 characters)

Items

| Name | Value | Summary |
| --- | --- | --- |
| Head | 0 | The head BodyPart. |
| Torso | 1 | The torso BodyPart. |
| LeftArm | 2 | The left arm BodyPart. |
| RightArm | 3 | The right arm BodyPart. |
| LeftLeg | 4 | The left leg BodyPart. |
| RightLeg | 5 | The right leg BodyPart. |
