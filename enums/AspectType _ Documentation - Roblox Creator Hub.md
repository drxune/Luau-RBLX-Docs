Engine Enum

AspectType

---

Used by [UIAspectRatioConstraint.AspectType](/docs/reference/engine/classes/UIAspectRatioConstraint#AspectType) to control sizing behavior.

Items

| Name | Value | Summary |
| --- | --- | --- |
| FitWithinMaxSize | 0 | The object will be the maximum size possible within its own [AbsoluteSize](/docs/reference/engine/classes/GuiBase2d#AbsoluteSize). |
| ScaleWithParentSize | 1 | The object's maximum size will be the size of the parent while still maintaining the aspect ratio. |
