Engine Enum

MeshType

---

The MeshType Enum is used to set what type of mesh the [SpecialMesh](/docs/reference/engine/classes/SpecialMesh) is.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Head | 0 | The mesh is the default [Humanoid](/docs/reference/engine/classes/Humanoid) head mesh. |
| Torso | 1 | The mesh is the default Humanoid torso mesh. |
| Wedge | 2 | The mesh is a wedge. |
| Sphere | 3 | The mesh is a sphere. |
| Cylinder | 4 | The mesh is a cylinder. |
| FileMesh | 5 | The mesh is determined by the Roblox mesh asset ID provided. |
| Brick | 6 | The mesh is a brick (just like the shape of a default [BasePart](/docs/reference/engine/classes/BasePart)). |
| Prism | 7 | Deprecated |
| Pyramid | 8 | Deprecated |
| ParallelRamp | 9 | Deprecated |
| RightAngleRamp | 10 | Deprecated |
| CornerWedge | 11 | Deprecated |
