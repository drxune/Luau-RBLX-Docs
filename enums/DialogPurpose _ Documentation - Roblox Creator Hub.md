Engine Enum

DialogPurpose

---

The DialogPurpose enum is used to set the icon of a [Dialog](/docs/reference/engine/classes/Dialog).

Items

| Name | Value | Summary |
| --- | --- | --- |
| Quest | 0 | Shows an exclamation point ('''!'''). |
| Help | 1 | Shows a question mark ('''?'''). |
| Shop | 2 | Shows a dollar sign ('''$'''). |
