Engine Enum

IKControlConstraintSupport

---

Values for [Workspace.IKControlConstraintSupport](/docs/reference/engine/classes/Workspace#IKControlConstraintSupport). Sets the support for
constraints for [IKControls](/docs/reference/engine/classes/IKControl) in your experience.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Default | 0 | The default option for [IKControls](/docs/reference/engine/classes/IKControl). The default is currently set to Enabled. |
| Disabled | 1 | Disables support for constraints for [IKControls](/docs/reference/engine/classes/IKControl). When disabled, [IKControls](/docs/reference/engine/classes/IKControl) still work, but ignore physics constraints. |
| Enabled | 2 | Enables support for constraints for [IKControls](/docs/reference/engine/classes/IKControl). |
