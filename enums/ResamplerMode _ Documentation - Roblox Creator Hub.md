Engine Enum

ResamplerMode

---

Determines the image filtering used.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Default | 0 | Bilinear filtering of the four near pixels in the image. |
| Pixelated | 1 | Nearest neighbor filtering of the closest pixel in the image. |
