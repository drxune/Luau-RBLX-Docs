FunctionalTestResult

---

Represents the outcome of a single functional test run.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Passed | 0 | The test run was successful. |
| Warning | 1 | The test run is potentially unstable. |
| Error | 2 | The test run failed. |
