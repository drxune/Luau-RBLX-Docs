Engine Enum

ChatStyle

---

The ChatStyle Enum is used to set the style of [Chat](/docs/reference/engine/classes/Chat) used in a game via
the [Players:SetChatStyle()](/docs/reference/engine/classes/Players#SetChatStyle) method.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Classic | 0 | Displays chats in the chat user interface at the top-left corner of the screen. |
| Bubble | 1 | Displays chats in a bubble above the sender's head. |
| ClassicAndBubble | 2 | Has the effects of Classic and Bubble combined. |
