Engine Enum

ZIndexBehavior

---

Used by [LayerCollector.ZIndexBehavior](/docs/reference/engine/classes/LayerCollector#ZIndexBehavior) to control how the
[GuiObject.ZIndex](/docs/reference/engine/classes/GuiObject#ZIndex) property behaves for descendants.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Global | 0 | The ZIndex property overrides the default value computed from the depth in the hierarchy. |
| Sibling | 1 | The ZIndex property controls the order in which the GuiObject renders relative to its siblings. |
