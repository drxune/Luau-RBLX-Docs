Engine Enum

VirtualCursorMode

---

Enables Virtual Cursor mode within an experience. By default, this property is
disabled.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Default | 0 | The Virtual Cursor is in the default state. |
| Disabled | 1 | The Virtual Cursor is disabled. |
| Enabled | 2 | The Virtual Cursor is enabled. |
