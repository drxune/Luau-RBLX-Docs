Engine Enum

FontStyle

---

Describes whether a [Font](/docs/reference/engine/datatypes/Font) style is normal or italic. Used by
[Font.Style](/docs/reference/engine/datatypes/Font#Style).

Items

| Name | Value | Summary |
| --- | --- | --- |
| Normal | 0 | The text renders normally with this font. |
| Italic | 1 | The text renders italic with this font. |
