Engine Enum

StrokeSizingMode

---

Used by [UIStroke.StrokeSizingMode](/docs/reference/engine/classes/UIStroke#StrokeSizingMode) to determine whether the stroke's
[Thickness](/docs/reference/engine/classes/UIStroke#Thickness) will be measured in pixels or be relative
to the parent.

Items

| Name | Value | Summary |
| --- | --- | --- |
| FixedSize | 0 | [Thickness](/docs/reference/engine/classes/UIStroke#Thickness) is measured in pixels. |
| ScaledSize | 1 | [Thickness](/docs/reference/engine/classes/UIStroke#Thickness) is relative to minimum parent width or height. If stroke is on text, thickness is relative to font size. |
