Engine Enum

OrientationAlignmentMode

---

The OrientationAlignmentMode Enum is used to select the number of
[Attachments](/docs/reference/engine/classes/Attachment) used in an alignment.

Items

| Name | Value | Summary |
| --- | --- | --- |
| OneAttachment | 0 | Use one attachment. |
| TwoAttachment | 1 | Use two attachments. |
