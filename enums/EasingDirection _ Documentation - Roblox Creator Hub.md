Engine Enum

EasingDirection

---

These enum values are passed to [TweenInfo.new()](/docs/reference/engine/datatypes/TweenInfo#new) to control the
direction [Tweens](/docs/reference/engine/classes/Tween) play in.

Items

| Name | Value | Summary |
| --- | --- | --- |
| In | 0 | The easing style is applied in a forward direction. |
| Out | 1 | The easing style is applied in a reverse direction. |
| InOut | 2 | The easing style is applied forward for the first half and in reverse for the second half. |
