Engine Enum

FriendRequestEvent

---

Describes the event when a player sends or responds to a connection request.
This is used by the in-game connection request system.

Items

| Name | Value | Summary |
| --- | --- | --- |
| Issue | 0 | A player invokes (sends) a connection request. |
| Revoke | 1 | A player revokes (cancels) a previously invoked (sent) connection request. |
| Accept | 2 | A player accepts a connection request. |
| Deny | 3 | A player denies a connection request. |
