ThrottlingPriority

---

Controls the amount of throttling under the [HttpRbxApiService](/docs/reference/engine/classes/HttpRbxApiService).

Items

| Name | Value | Summary |
| --- | --- | --- |
| Default | 0 | Standard throttling. |
| ElevatedOnServer | 1 | Less throttling if on the server. |
| Extreme | 2 | No throttling, absolutely no exceptions. |
